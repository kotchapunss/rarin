
# Self-Calculator • Minimal Brown (React + Vite + Tailwind)

A minimal, pastel-brown themed UI replicating the flow of type → package → details → add-ons with a responsive sidebar summary.

## Features
- **Multi-language Support**: Toggle between Thai and English
- Stepper with Back/Next
- Type selection: wedding, event, photo
- Package select with expandable details per package
- Guest count + period time (morning/evening) + notes
- Add-ons per type
- Responsive Summary: sidebar on desktop, sticky bottom on mobile

## Language Toggle
The application supports both Thai (ไทย) and English languages. Click the language toggle button in the header (globe icon with current language) to switch between languages. All content including:
- Navigation and buttons
- Package names and descriptions  
- Form labels and placeholders
- Add-on names
- Summary labels

## Quick start
```bash
npm install
npm run dev
```
Build for production:
```bash
npm run build
npm run preview
```
